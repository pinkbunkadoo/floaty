What's all this?
Floaty is an app for turning images into transparent desktop windows.

Getting started
  1. Drag one or more images onto the drop window
  2. Make the desired adjustments to any images
  3. Tap the eye icon or press slash (/) on the keyboard to enter covert mode*
  4. Tap the tray icon to leave covert mode
* In covert mode images are non-interactive and appear above all other desktop windows
Global shortcuts
Shortcut Description
Slash (/) Covert mode.
Ctrl + QQuit the app.

Shortcut Description
